# Data-Classification-Using-AI
A Machine Learning classification project developed using Python and Scikit-Learn to classify Iris flower species based on their characteristics.

This project was developed as part of my Internship Program.

The goal of this project is to build a simple Machine Learning classification model that can predict the species of a flower using the Iris dataset.

The model learns from existing data and then classifies new flower samples into the correct category.

---

## Project Objective

- Load and understand a dataset
- Split data into training and testing sets
- Train a Machine Learning model
- Evaluate model accuracy
- Predict flower species

---

## Technologies Used

- Python
- Scikit-Learn
- Pandas
- Matplotlib
- VS Code

---

## Dataset

This project uses the famous Iris Dataset which contains:

- 150 flower samples
- 4 flower features
- 3 flower species

## Species

- Setosa
- Versicolor
- Virginica

---

## Machine Learning Algorithm

Decision Tree Classifier

The model learns patterns from flower measurements and predicts the correct flower species.
